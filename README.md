# Npminit

> My coffee-script cross-platform skeleton

## Installation
```bash
$ npm install 59naga/npminit --global
```

## Generate project
```bash
$ npminit

# name:
foo-bar-baz

# description:
kaboom

# Generating the foo-bar-baz .................. done
# foo-bar-baz
# ├─ README.md
# ├─ bower.json
# ├─ index.coffee
# ├─ node_modules
# │  ├─ ghooks
# │  ├─ jasminetea
# │  └─ object-parser-cli
# ├─ package.json
# ├─ src
# │  └─ index.coffee
# └─ test
#     └─ api.spec.coffee
# 
# Generated /Users/59naga/Downloads/foo-bar-baz
```
